# 珠海阳光儿童用品有限公司官网

## Sunnylove Baby Products Zhuhai Co., Ltd.

这是一个现代化、响应式的企业官网，采用黄色为主色调，展示珠海阳光儿童用品有限公司的品牌形象和企业实力。

**最后更新时间：2026年3月12日**

## 项目特点

- ✨ 现代化设计，以温暖的黄色为主色调
- 📱 完全响应式，支持各种设备
- 🎨 流畅的动画效果和交互体验
- 🚀 轻量级，无需复杂框架
- 🌐 中英双语展示

## 网站结构

1. **首页 Hero Section** - 品牌展示与核心口号
2. **公司概述** - 公司介绍、实力、技术优势等
3. **品牌理念** - 品牌口号、使命、愿景、核心价值观
4. **产品展示** - 婴儿推车、安全座椅、高餐椅等产品系列
5. **公司优势** - 六大核心竞争力展示
6. **未来展望** - 战略目标与发展路径
7. **联系我们** - 公司地址、电话、网址

## 技术栈

- HTML5
- CSS3 (使用 CSS Grid 和 Flexbox)
- Vanilla JavaScript
- Google Fonts (Noto Sans SC)

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器运行项目

## 浏览器支持

- Chrome (推荐)
- Firefox
- Safari
- Edge
- 移动端浏览器

## 响应式断点

- 桌面端: > 768px
- 移动端: ≤ 768px

## 功能特性

- 固定导航栏，滚动时有阴影效果
- 移动端汉堡菜单
- 平滑滚动效果
- 卡片悬停动画
- 滚动触发的淡入动画
- 返回顶部按钮

## 自定义修改

### 修改颜色
在 `styles.css` 文件的 `:root` 部分修改 CSS 变量

### 修改内容
直接编辑 `index.html` 文件中的文本内容

### 添加产品图片
在产品展示部分添加 `<img>` 标签，并将图片放在项目目录中
